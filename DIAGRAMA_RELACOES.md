## 🔑 Chave das Relações

### Legenda:
- **PK** = Primary Key (chave primária)
- **FK** = Foreign Key (chave estrangeira)
- **[1:N]** = Um para Muitos
- **[N:M]** = Muitos para Muitos
- **STRING!** = Importante: código é string, não int
